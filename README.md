# CS32-FP
CS 32 Final Project: Rules-based Golf Shot Optimizer

This is a rules-based golf shot optimizer that helps a user select the appropriate club for a given shot. The program will take as input the user’s average yardages for each club, the distance to the target, and environmental factors such as headwind or tailwind. Using simple, predefined rules, the program will adjust the effective playing distance based on these conditions and recommend the best club to use.

The system may also suggest how to play the shot (e.g., a normal swing, taking distance off, or adding spin) depending on the conditions and the adjusted yardages. The goal is to implement a clear algorithm that processes inputs and produces a practical recommendation for club selection and shot type.
